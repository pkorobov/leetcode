### 287. Find the Duplicate Number

Link: https://leetcode.com/problems/find-the-duplicate-number

Complexity: medium

Two constraints:
- Do not modify the array
- Use constant extra space only

Solution 1: \
Binary search \
Complexity: O(n * log(n))

Solution 2: \
Slow and fast pointers (Floyd's algorithm) \
Complexity: O(n)
